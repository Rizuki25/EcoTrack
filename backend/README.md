# Backend

Backend dibuat setelah baseline YOLO tervalidasi.

Ruang lingkup MVP:

- Autentikasi dan peran pengguna/petugas/admin.
- Validasi QR pengguna.
- Prediksi kategori sampah.
- Transaksi setoran dan detail item.
- Perhitungan poin berdasarkan kategori dan berat.
- Riwayat, reward, dan audit log.

Rekomendasi awal: FastAPI + PostgreSQL/Supabase.
